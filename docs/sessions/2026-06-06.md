## Notes for your records

**Confirmed bugs squashed this session:**
- Animation Studio: synchronous frame switching, no more disappearing content
- Cauldron Café: minus button actually subtracts now (Phaser display-order issue + 7-yr-old tap-target ergonomics)

**Architecture additions:**
- `/api/humphrey-observe` — new Vercel serverless endpoint that calls Claude Haiku 4.5 vision (`claude-haiku-4-5-20251001`). Costs roughly $0.0005 per observation. With Nigel's typical 30-min session firing maybe 30 observations, that's ~$0.015 per session — negligible.
- `js/humphrey-observer.js` — client module that orchestrates throttling, autopilot polling, and handoff to the Humphrey speech engine.

**Patterns learned (for future builds):**
- Phaser hit-test priority follows display depth, not creation order. Always `setDepth()` interactive children inside larger hit zones.
- Touch targets need to be ≥56px for 7-yr-old fingers. Visible button can stay smaller; layer an invisible larger hit overlay on top.
- Synthetic pointer events don't reach Phaser's input queue in sandbox — physical device verification stays in the loop.
